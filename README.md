# One Page Advisor

## Description
OPA (One Page Advisor) is a tool that aggregates the results of trusted advisor checks across multiple AWS accounts.

## Usage
1. Specify the AWS profile names in the configuration file [config.py](./config.py).    
2. Run the tool.    
```
usage: python opa.py
```
